Requirements:
- PHP >= 5.1.0
- MySQL >= 5
- At least 1 GB hard space (for one season match reports and MySQL data)
- OS: Linux or Windows

Recommendations:
- OS: Linux

Instalation:
- Upload source
- Set permissions of config.php to 0777
- Run install.php
- Fill the required information (database login data)
- Press "Install game"
- Wait for the game to be fully installed
- Delete "install.php"
- Login with username "admin", password "adminadmin"
- Enjoy!
